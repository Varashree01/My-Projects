#QR Code Generator

This is a simple Python script that generates a QR code from a user entered URL and saves it as an image file. It uses the qrcode library to create the QR code.

Features :
Accepts any URL as input
Generates a QR code image
Saves the output to a specified file path
Easy to run and modify

Requirements :
Install the required library:
pip install qrcode

How to Use :
Run the script in a Python environment.
Enter the URL when prompted.
The script creates a QR code and saves it to the path defined in the code.

Output :
The generated QR code is saved as qrcode.png in the given file path.
